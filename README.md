# ButterKnifeDemo:

# FaceDemo:
使用face++,进行人脸识别,将两个人的头像进行比较,返回相似度.
首先需要在http://www.faceplusplus.com.cn/注册账号,添加项目,得到API Key和API Secret.

# HelperDemo:
记录了一些常用的方法和一些常用的帮助类.

# JsonDemo:

# LeakCanary:
检测所有的内存泄漏,先声明application,调用RefWatcher的watch();查看制定位置或者对象的内存情况.

# PicassoDemo:
用于对图片的加载与缓存,也可以直接从网上档取图片.

# PullParseXmlDemo:

# SignaturePadDemo:

# SoapDemo:

# SugarDemo:

# WebViewAndJSDemo:
